# Journey of Innovation – Magazine Style Web Page

A modern magazine-style article web page built using **HTML, CSS, and JavaScript**.  
This project demonstrates a clean two-column layout, responsive images, and a simple front-end project structure.

---

## 📌 Project Description
Journey of Innovation is a simple front-end project designed to showcase a magazine-style article layout.  
It uses modern web design principles to create a clean and readable interface.

The project highlights:
- Structured HTML layout
- Responsive CSS design
- Simple JavaScript interaction

---

## 🚀 Features

- 📄 Magazine-style two-column article layout  
- 🎨 Clean and modern typography  
- 🖼 Responsive images  
- ⚡ Lightweight and fast loading  
- 📱 Basic responsive design  

---

## 🛠 Technologies Used

- CSS3  
- JavaScript
